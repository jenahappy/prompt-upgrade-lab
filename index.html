🧩 P-C-T-C 공식과 5단계 업그레이드 전략
좋은 프롬프트는 한 번에 완성되지 않습니다. 5단계를 거쳐 점진적으로 완성됩니다.
🎭
P — Persona
역할 정의
AI에게 어떤 전문가로 답변할지 명확히 지정합니다. 역할이 바뀌면 답변의 수준과 관점이 완전히 달라집니다.
🌐
C — Context
배경 상황
AI는 당신의 상황을 모릅니다. 왜 이 작업이 필요한지, 목표가 무엇인지, 대상이 누구인지 설명해야 합니다.
🎯
T — Task
구체적 임무
"써줘"가 아닌 "작성해줘 / 분석해줘 / 요약해줘"처럼 단일하고 명확한 행동 동사로 지시합니다.
📐
C — Constraint
제약 조건
분량, 형식, 말투, 금지 표현, 반드시 포함할 키워드 등을 구체적으로 지정합니다. AI의 가드레일입니다.
💡
FS — Few-shot
예시 학습
원하는 결과물 샘플을 1~2개 보여주는 기술. "이런 스타일로"라는 설명보다 예시 1개가 10배 정확합니다.
