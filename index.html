📖 공식 이해

P-C-T-C 공식 완전 정복

좋은 프롬프트는 4가지 요소로 구성됩니다. 5단계를 거쳐 점진적으로 완성됩니다.

🎭
P — PERSONA
역할 정의
AI에게 어떤 전문가로 답변할지 지정합니다. 역할이 달라지면 답변 수준이 완전히 달라집니다.
🌐
C — CONTEXT
배경 상황
AI는 당신의 상황을 모릅니다. 왜 필요한지, 목표가 무엇인지, 대상이 누구인지 설명해야 합니다.
🎯
T — TASK
구체적 임무
"써줘"가 아닌 "작성해줘 / 요약해줘 / 표로 정리해줘"처럼 단일하고 명확한 동사로 지시합니다.
📐
C — CONSTRAINT
제약 조건
분량, 형식, 말투, 금지 표현을 지정합니다. AI가 엉뚱하게 흘러가지 않도록 막는 가드레일입니다.
💡
FS — FEW-SHOT
예시 학습
원하는 결과물 샘플 1~2개를 보여주는 기술입니다. 설명 10번보다 예시 1개가 10배 더 정확합니다.

💼 취업희망자

이력서부터 면접까지 — AI로 취업 경쟁력 높이기

자기소개서 작성을 예시로, Lv.1 → Lv.5 단계별 업그레이드를 직접 경험해보세요.

📋 실습 시나리오 경영학 전공 김지원(27세)은 대학 마케팅 동아리에서 2년간 인스타그램을 운영해 팔로워를 3,000명 → 12,000명으로 성장시켰습니다. 친환경 브랜드 ㈜그린라이프의 디지털 마케팅 직무에 지원하며, "지원동기와 입사 후 포부"(700자 이내)를 작성해야 합니다.
Lv.1
🔴 제로샷 — 그냥 던지기
★☆☆☆☆ ▼
❌ 왜 나쁜가?
  • AI가 지원자의 전공·경험·회사명을 전혀 모름
  • "열정", "도전", "성장"으로 가득 찬 뻔한 문장 생성
  • 실제 제출하려면 처음부터 다시 써야 함
❌ 나쁜 프롬프트
마케팅 회사 자기소개서 지원동기 써줘.
Lv.2
🟠 역할(P)만 추가
★★☆☆☆ ▼
⚠️ 개선됐지만 여전히 부족
  • 역할은 줬지만 지원자의 실제 정보가 없음
  • 어떤 회사인지, 어떤 경험을 강조해야 할지 AI가 추측해서 씀
너는 10년 경력의 취업 컨설턴트야. 마케팅 회사 자기소개서의 지원동기와 입사 후 포부를 써줘.
Lv.3
🟡 P + T 조합 — 역할 + 임무
★★★☆☆ ▼
⚠️ 방향은 맞지만 맥락이 빠짐
  • 역할과 임무는 있지만 지원자의 실제 경험 정보가 없음
  • 회사 특징, 직무 핵심을 AI가 임의로 만들어냄 (환각 위험)
너는 10년 경력의 취업 컨설턴트야. 경영학 전공, SNS 운영 경험이 있는 27세 취업희망자의 마케팅 직무 자기소개서 '지원동기와 입사 후 포부' 항목 초안을 작성해줘.
Lv.4
🟢 P-C-T-C 완성형 — 4요소 전부 ⭐ 권장
★★★★☆ ▼
✅ 왜 좋은가?
  • 4가지 요소가 모두 있어 AI가 추측할 것이 거의 없음
  • 실제 경험(수치 포함)을 넣어 개인화된 결과물이 나옴
  • 분량·말투·금지어까지 지정해 바로 제출 가능한 초안이 나옴
① 역할 (P) 너는 대기업 인사팀 출신 10년 경력의 취업 컨설턴트야. 지원자의 강점을 살리며 진정성 있는 자소서를 쓰는 전문가야. ② 배경 (C) - 이름: 김지원 (27세) - 전공: 경영학 (마케팅 부전공) - 경험: 마케팅 동아리 인스타그램 운영 2년 (팔로워 3,000명 → 12,000명으로 성장) - 지원 회사: ㈜그린라이프 (친환경 라이프스타일 브랜드) - 지원 직무: 디지털 마케팅 (SNS · 콘텐츠 기획) ③ 임무 (T) 위 정보를 바탕으로 자기소개서 '지원동기와 입사 후 포부' 항목의 초안을 작성해줘. ④ 조건 (C) - 분량: 650~700자 이내 - 말투: 진솔하고 구체적, 과장 금지 - 동아리 SNS 성장 수치(3,000→12,000)를 반드시 포함 - "열정", "도전", "성장" 같은 진부한 단어 절대 사용 금지 - 입사 후 3년 내 달성 목표를 구체적으로 언급할 것
Lv.5
🔵 마스터 — P-C-T-C + 문체 예시(Few-shot)
★★★★★ ▼
🏆 Lv.4보다 강력한 이유
  • 내가 원하는 문체 샘플을 직접 보여줘 AI가 패턴으로 학습함
  • 말투·문장 길이·시작 방식까지 정확하게 재현됨
  • 첫 번째 결과물이 곧바로 제출 가능한 수준이 됨
① 역할 (P): 대기업 인사팀 출신 10년 경력 취업 컨설턴트 ② 배경 (C): 김지원(27세), 경영학 전공 동아리 SNS 팔로워 3,000→12,000 성장 지원: ㈜그린라이프 / 디지털 마케팅 ③ 임무 (T): '지원동기와 입사 후 포부' 자소서 초안 작성 ④ 조건 (C): 700자 이내, 수치 포함, 진부한 단어 절대 금지 ⑤ 문체 예시 (Few-shot) [내가 원하는 스타일 예시야 — 이 패턴을 학습해줘] "처음 동아리 SNS를 맡았을 때, 팔로워는 3,000명이었습니다. 저는 숫자를 바꾸기 위해 데이터를 분석했습니다. 게시 시간, 해시태그, 썸네일 색상까지 하나씩 테스트했고, 6개월 만에 12,000명이 됐습니다." → 핵심 특징: 짧은 문장, 구체적 행동 동사, 수치로 결과 증명 이제 위 스타일 그대로 지원동기와 포부 전체를 완성해줘.

📝 추가 연습 — 면접 준비 프롬프트 (Lv.4)

면접관이 SNS 운영 경험과 데이터 분석 능력을 중점적으로 볼 것이라고 공고에 명시되어 있습니다. 예상 질문 5개와 STAR 기법 답변 초안을 AI로 미리 준비하고 싶습니다.
① 역할 (P) 너는 대기업 인사팀에서 10년간 면접관을 담당한 HR 전문가야. ② 배경 (C) - 지원자: 김지원 (27세, 경영학 전공) - 지원 직무: 디지털 마케팅 (SNS·콘텐츠 기획) - 지원 회사: ㈜그린라이프 - 핵심 경험: 동아리 인스타그램 팔로워 3,000→12,000 성장 - 면접 포인트: SNS 운영 경험, 데이터 분석 능력 ③ 임무 (T) 1차 직무면접 예상 질문 5개와 STAR 기법 모범 답변 초안을 작성해줘. ④ 조건 (C) - 질문은 직무 역량 관련으로만 구성 (인성 질문 제외) - STAR 기법: 상황(Situation) → 행동(Action) → 결과(Result) 순서 - 답변 분량: 질문당 200자 내외 - 동아리 SNS 경험을 2개 이상의 답변에 활용할 것

① 역할 (P): 고객 서비스 및 리뷰 관리 전문가 ② 배경 (C) - 가게: 부산 해운대 수제버거 가게 '버거퀸즈' - 실제 리뷰 내용 (별점 2점): "웨이팅이 너무 길고, 직원이 불친절했어요. 버거 맛은 괜찮았지만 다시 오고 싶지 않네요." - 실제 상황: 주말 웨이팅 30분 이상 발생, 아르바이트생 교육 중 ③ 임무 (T): 위 부정 리뷰에 대한 사장님 공식 답변글을 작성해줘. ④ 조건 (C) - 분량: 150~200자 이내 - 변명하거나 반박하지 말 것 - 불편함을 인정하고 진심으로 사과할 것 - 개선 의지를 구체적으로 언급할 것 (직원 교육, 웨이팅 시스템) - 재방문을 유도하는 따뜻한 마무리 문장 포함

💡 핵심 인사이트 5가지

① Lv.1의 진짜 문제는 '모름'이다

막연하게 쓰는 이유는 프롬프트 작성법을 몰라서입니다. 4가지 요소(P-C-T-C)를 알면 누구나 Lv.4 이상을 쓸 수 있습니다.

② Lv.4의 핵심은 '조건(C — Constraint)'이다

역할·배경·임무가 있어도 형식·분량·금지어를 지정하지 않으면 AI는 자유롭게 채워 넣습니다. 제약 조건이 결과물의 품질을 결정합니다.

③ Lv.5의 비결은 '설명 → 보여주기' 전환이다

"친근한 말투로 써줘"(설명)보다 실제 예시 1개(보여주기)가 10배 정확합니다. Show, don't tell 원칙은 AI에게도 그대로 적용됩니다.

④ 좋은 프롬프트는 '재사용 자산'이다

Lv.4~5 프롬프트를 한 번 완성해두면 다음엔 내용만 바꿔 반복 사용할 수 있습니다. 나만의 프롬프트 라이브러리를 만들면 업무 속도가 2~3배 빨라집니다.

⑤ 결과물이 마음에 들지 않으면 AI가 아닌 프롬프트를 먼저 점검하라

AI는 '막연한 질문 = 막연한 답변'의 원칙을 따릅니다. 결과물이 마음에 들지 않을 때 90%는 프롬프트 문제입니다. AI 탓을 하기 전에 P-C-T-C를 먼저 확인하세요.
